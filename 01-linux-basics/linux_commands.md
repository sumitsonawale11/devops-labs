# Linux Basics

## What is Linux?

Linux is an open-source Unix-like operating system kernel.

Linux distributions include:

- Ubuntu
- Debian
- Rocky Linux
- Amazon Linux
- CentOS

## Why Linux?

- Open Source
- Stable
- Secure
- Fast
- Lightweight
- Used in Cloud
- Used in Servers
- Supports Automation

## Linux Architecture

User
↓
Applications
↓
Shell
↓
Kernel
↓
Hardware

### Shell Examples

- bash
- sh
- zsh

### Kernel Responsibilities

- Process Management
- Memory Management
- File System
- Networking
- Device Drivers

## Linux File System

### /

Root directory of Linux.

### /home

Home directories of normal users.

Examples:

- /home/ubuntu
- /home/sumit

### /root

Home directory of the root user.

### /etc

Configuration files.

Examples:

- /etc/nginx
- /etc/ssh
- /etc/php

### /var

Variable data.

Examples:

- /var/log
- /var/lib/docker

### /tmp

Temporary files.

### /opt

Optional third-party software.

### /usr

User programs and binaries.

### /bin

Essential Linux commands.

### /dev

Device files.
